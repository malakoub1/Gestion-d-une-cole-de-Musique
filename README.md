# 🎵 Gestion d’une École de Musique

## 📌 Description
Ce projet est une application web développée avec **Spring Boot**, **Spring Data JPA** et **Thymeleaf**.  
Il permet de gérer une école de musique à travers :
- les instruments
- les professeurs
- les cours programmés

L’application propose également des fonctionnalités de filtrage et de statistiques.

---

## ⚙️ Technologies utilisées
- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA
- Thymeleaf
- MySQL (ou H2)
- Maven

---

## 🧩 Fonctionnalités

### 🎸 Gestion des instruments
- Ajouter, modifier, supprimer un instrument
- Suivre la disponibilité
- Filtrer par famille (cordes, vents…)

### 👨‍🏫 Gestion des professeurs
- Ajouter, modifier, supprimer un professeur
- Associer une spécialité

### 📅 Gestion des cours
- Planifier un cours (jour, heure, durée, niveau)
- Associer un professeur
- Filtrer les cours par jour et niveau

### 📊 Statistiques
- Nombre total de cours
- Nombre d’instruments disponibles / indisponibles
